# Platform Specs — 平台素材规格

输出平台物料尺寸、格式、安全区和交付要求。平台规则经常变化；给生产团队使用前要优先核验官方或平台帮助中心信息。

## 使用原则

1. 如果用户要实际投放或交付生产，必须确认平台、广告位、素材类型和发布日期。
2. 涉及最新规格、审核规则、广告位限制、文件大小、时长、禁投规则时，联网检索官方来源。
3. 如果无法确认官方来源，标注“待平台确认”，不要把旧规格当成确定事实。
4. 输出不仅写尺寸，也写安全区、裁切风险、首帧/封面、字幕、Logo、CTA 和可交付文件命名建议。

## 常用规格速查

以下为常见起点，交付前仍需按平台和广告位验证：

| 平台 | 常见物料 | 常见规格 | 备注 |
|---|---|---|---|
| 抖音 | 竖版短视频 | 1080x1920, 9:16, MP4/MOV | 注意上下 UI 安全区、首 3 秒信息 |
| 抖音 | 横版视频 | 1920x1080, 16:9 | 适合部分信息流/横版场景 |
| 小红书 | 图文封面/内页 | 1080x1440, 3:4 | 封面文字少而清晰，注意关键词 |
| 小红书 | 竖版视频 | 1080x1920, 9:16 | 封面影响点击 |
| 微信公众号 | 头图/封面 | 常用 900x383 等比例 | 重点元素居中，避免裁切 |
| 朋友圈 | 图片/视频 | 1:1、3:4、9:16 常见 | 需确认广告位和投放后台要求 |
| 视频号 | 竖版视频 | 1080x1920, 9:16 | 关注标题、封面、直播承接 |
| 微博 | 配图/视频 | 竖图、方图、横视频均可 | 话题页和信息流展示裁切不同 |
| B站 | 视频/封面 | 1920x1080, 16:9 | 封面信息密度和标题配合 |
| 淘宝/天猫 | 主图/详情页 | 主图常用 800x800，详情页常用 750 宽 | 以店铺后台要求为准 |
| 京东 | 主图/详情页 | 主图常用 800x800，详情页按后台 | 以商家后台和类目规则为准 |

## 规则

- 不同广告位不要混用规格；开屏、信息流、搜索、直播间、店铺素材可能不同。
- 安全区优先保守处理，避免按钮、字幕、价格和 Logo 被平台 UI 遮挡。
- 若素材将用于广告投放，提醒同步核对平台审核规则和行业资质要求。
