# Competitor Analysis — 竞品研究

系统研究指定竞品或品类玩家，输出能直接服务策略、创意、内容、KOL 和媒介决策的竞争洞察。

## 研究原则

1. 涉及近期动作、投放、价格、KOL、平台表现时必须联网检索，并标明来源和日期。
2. 区分直接竞品、替代竞品、学习对象和非同量级标杆。
3. 不编造投放量、达人姓名、销售数据、用户评价或 Campaign 成效。
4. 市场空白必须由对比得出，不能凭感觉判断。
5. 结论要落到“我们应该避开/借鉴/主攻什么”。

## 竞品选择

- 客户指定：优先分析客户点名对象。
- 未指定：选 3-5 个代表性玩家，覆盖头部、同价格带、同人群、替代方案。
- 深度策划：选 2-3 个核心竞品做内容/KOL/渠道拆解。
- 标杆学习：可加入 1 个跨品类标杆，但明确其不可直接类比的部分。

## 信息收集维度

| 维度 | 观察内容 | 推荐来源 |
|---|---|---|
| 品牌定位 | 价值主张、目标人群、差异化、品牌语气 | 官网、旗舰店、品牌号 |
| 产品与价格 | Hero SKU、价格带、促销机制、组合 | 电商平台、官方商城 |
| 内容策略 | 主题、视觉、话术、内容支柱、爆款结构 | 抖音、小红书、微博、B站、公众号 |
| 渠道组合 | 社媒、搜索、电商、直播、私域、线下 | 公开账号、平台搜索、媒体报道 |
| KOL 生态 | 达人层级、垂类、人设、合作内容 | 公开发布内容、达人主页、榜单工具 |
| 用户反馈 | 正负面评价、常见疑虑、购买理由 | 小红书、电商评论、问答社区 |
| 近期动作 | Campaign、大促、联名、代言、话题 | 新闻、社媒、官方公告 |
| 威胁判断 | 对本品牌的挤压、优势、弱点 | 综合分析 |

## 分析框架

### 1. 竞争分层

| 类型 | 判断标准 | 策略意义 |
|---|---|---|
| 直接威胁 | 同人群、同价格带、同渠道 | 必须差异化 |
| 价格战对手 | 低差异、高促销 | 避免陷入同质化 |
| 心智标杆 | 强品牌资产或强内容心智 | 可学习表达方式 |
| 空白参照 | 未满足某场景/人群 | 可作为突破口 |

### 2. 可攻击空白

从以下角度找机会：
- 人群空白：竞品忽视的细分人群。
- 场景空白：用户真实使用/购买场景未被表达。
- 证据空白：竞品只讲情绪，缺少可信证明。
- 内容空白：平台已有需求但品牌内容供给不足。
- 渠道空白：竞品强内容弱转化，或强投放弱种草。

## 规则

- 每个关键判断都附来源、观察样本或明确推断依据。
- 用户原话必须来自真实公开评论；无法取得时写“未验证，不引用原话”。
- 不把“竞品做得多”直接等同于“竞品做得好”；要说明效果证据或仅标注为可见动作。
