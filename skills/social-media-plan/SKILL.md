# Social Media Plan — 社交媒体内容方案

制定跨平台社交内容策略，让每个平台承担清楚角色，而不是把同一条内容复制到所有平台。

## 规划原则

1. 先定义平台角色，再写选题。
2. 每条内容都要服务漏斗环节：认知、兴趣、搜索、信任、转化、复购。
3. 平台语言必须差异化，但核心策略和卖点保持一致。
4. 内容日历要考虑制作难度、素材复用、审核周期和发布节奏。
5. 当涉及平台算法、规格或广告位规则时，需验证当前规则。

## 平台角色速查

| 平台 | 用户心态 | 适合任务 | 内容重点 | 常见指标 |
|---|---|---|---|---|
| 抖音 | 刷到即看、即时转化 | 爆发、直播、短链路成交 | 前 3 秒、场景、强钩子 | 完播、CTR、CVR、GMV |
| 小红书 | 搜索种草、理性比较 | 种草、口碑、搜索承接 | 标题、封面、真实体验、关键词 | 收藏、搜索、评论、站外转化 |
| 微信公众号 | 深度阅读、信任关系 | 品牌背书、私域沉淀 | 观点、故事、价值解释 | 阅读、转发、线索 |
| 视频号 | 熟人/半熟人扩散 | 私域联动、直播承接 | 人设、信任、转发 | 观看、转发、直播预约 |
| 微博 | 热点讨论、公共情绪 | 话题扩散、事件营销 | 话题、互动、明星/IP | 话题阅读、互动 |
| B站 | 深度内容、圈层文化 | 专业解释、测评、口碑 | 信息密度、真实感、弹幕互动 | 完播、收藏、投币 |

## 内容支柱设计

| 支柱 | 作用 | 常见形式 |
|---|---|---|
| 问题教育 | 建立需求 | 科普、误区、场景痛点 |
| 证据证明 | 建立信任 | 测评、对比、实验、专家/用户证言 |
| 场景代入 | 促进想象 | 日常 Vlog、前后对比、使用教程 |
| 社交认同 | 增强欲望 | UGC、达人体验、榜单、口碑合集 |
| 转化承接 | 推动下单 | 福利、组合、直播预告、购买指南 |
| 品牌资产 | 长期认知 | 品牌故事、价值观、幕后内容 |

## 规则

- 每个选题必须有具体标题、钩子或画面建议。
- 不用“高级感”“年轻化”这类空词替代平台策略。
- 内容 KPI 要匹配平台角色；种草内容不只看曝光，转化内容不只看点赞。
