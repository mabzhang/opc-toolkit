# Campaign Planning — 整合 Campaign 策划

制定从洞察、策略、创意主题到渠道执行的整合 Campaign 方案。输出要能被客户理解，也能被创意、媒介、内容、达人和投放团队拆解执行。

## 输入要求

优先读取或建立：
- Brief 需求卡：目标、受众、产品、预算、时间、限制。
- 竞品与消费者洞察：市场空白、购买障碍、内容触发点。
- 品牌资产：品牌语气、视觉资产、历史案例、禁区。
- 执行边界：预算、人力、素材周期、法务/平台审核周期。

## 七层策略模型

| 层级 | 要回答的问题 | 产出 |
|---|---|---|
| 背景与洞察 | 为什么现在要做 | 行业/品类/消费者/竞品关键判断 |
| 核心挑战 | 这次真正要解决什么 | 一句话挑战 |
| 策略目标 | 业务、品牌、内容、渠道目标如何排序 | 分层目标与 KPI 方向 |
| 策略方向 | 用什么逻辑赢 | 核心主张、差异化、证据链 |
| 创意主题 | 用户记住什么 | 主题、创意领地、内容支柱 |
| 传播架构 | 在哪里、由谁、用什么内容说 | 阶段、渠道、KOL、媒介组合 |
| 效果与复盘 | 怎样判断成功 | KPI、优化机制、复盘节点 |

## 创意主题评估

主题必须通过以下检查：
- 清晰：10-14 字内能被复述，必要时配解释句。
- 有根：来自消费者洞察或品牌资产，不是空泛押韵。
- 可延展：能拆出短视频、图文、直播、话题、搜索内容。
- 有差异：避开竞品高饱和表达。
- 能转化：能连接产品证据、购买理由或行动入口。
- 合规：不碰行业禁语、绝对化承诺和平台风险表达。

## 传播节奏

| 阶段 | 目标 | 常见动作 | 关键指标 |
|---|---|---|---|
| 准备期 | 资料、素材、达人、投放账户就绪 | 策略定稿、脚本、选号、素材制作 | 准时交付率 |
| 预热期 | 建立期待和搜索入口 | KOL 种草、话题预埋、悬念内容 | 收藏、搜索、互动 |
| 爆发期 | 放大声量和转化 | 全渠道发布、直播、电商承接 | 曝光、CTR、CVR、GMV |
| 转化期 | 承接犹豫人群 | 测评、对比、优惠、私域召回 | 加购、转化成本 |
| 长尾期 | 沉淀口碑与复购 | UGC、复盘内容、二次传播 | 复购、口碑内容量 |

## 规则

- 不写“全平台种草”这类空话；每个平台必须有角色和任务。
- 创意主题不能只好听，必须能拆成内容和转化动作。
- 预算未知时按资源优先级规划，不假造金额。
